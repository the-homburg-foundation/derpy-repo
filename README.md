# Derpy Repository

## 1. Purpose
`derpy-repo` hosts curated packages and maintains the official package repository for Derpy GNU/Linux.

## 2. Contents
- Curated package builds
- Repository metadata
- Signing keys for package integrity

## 3. Usage
This repository serves as the official source of Derpy GNU/Linux packages and updates.

## 4. Contribution
Contributions must:
- Follow packaging standards
- Be properly tested and signed
- Include clear documentation for any added packages
